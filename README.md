# Flight

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.2.0. Make sure the latest Node and Angular is installed on local machine. 

## Development server

Download the project. Run `npm install`. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Code scaffolding

Angular Material is used for form fields and table to display the data. Custom CSS styling is applied for better UI. Filter and Sort are applied on the displayed table. 

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.

## Test
